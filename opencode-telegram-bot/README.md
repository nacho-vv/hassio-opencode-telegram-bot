# OpenCode Telegram Bot

Control OpenCode from Telegram - Run and monitor AI coding tasks from your phone.

## Setup

1. Get a bot token from [@BotFather](https://t.me/BotFather)
2. Get your user ID from [@userinfobot](https://t.me/userinfobot)
3. Configure the addon with your token and user ID
4. Ensure OpenCode is running and accessible at the configured URL
